## 暂停条件（任何阶段）

出现以下任一情况立即暂停扩张并回滚 Feature Flags：

- [ ] 数据错乱：目标/任务/打卡数据不一致
- [ ] 越权访问：用户可读取他人数据
- [ ] 重复扣费：支付事件导致多次扣款
- [ ] 提醒轰炸：用户收到大量重复提醒
- [ ] AI 大面积失败：> 30% 失败率
- [ ] 上传安全：未扫描文件可下载

## 恢复条件
- [ ] 根因已定位并修复
- [ ] 自动化测试验证通过
- [ ] 手动验证受影响流程正常
